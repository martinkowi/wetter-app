Martin's Wetter App

Eine einfache Wetter-App gebaut mit Python und Streamlit.

Features
- Aktuelle Temperatur & gefühlte Temperatur
- Wetterbeschreibung mit Bild
- Luftfeuchtigkeit
- Windgeschwindigkeit
- Luftdruck
- Sonnenaufgang & Sonnenuntergang
- Karte

Technologien
- Python
- Streamlit
- OpenWeatherMap API
- Docker
- Railway
